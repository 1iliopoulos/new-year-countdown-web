# 🎆 New Year Countdown Web App

Σύγχρονη και responsive web εφαρμογή που εμφανίζει **σε πραγματικό χρόνο** τον χρόνο που απομένει μέχρι την **επόμενη Πρωτοχρονιά**.  
Αναπτύχθηκε με **καθαρό HTML, CSS και JavaScript**.

---

## ✨ Χαρακτηριστικά

- ⏳ Ζωντανή αντίστροφη μέτρηση μέχρι την Πρωτοχρονιά
- 🔁 Αυτόματη προσαρμογή κάθε χρόνο (χωρίς hardcoded ημερομηνίες)
- 📱 Πλήρως responsive σχεδίαση (desktop & mobile)
- 🎨 Σύγχρονο dark UI με gradients
- ⚡ Ελαφριά και γρήγορη εφαρμογή (vanilla stack)
- 🧩 Εύκολη ενσωμάτωση σε άλλα sites ή projects

---

## 🛠️ Τεχνολογίες

- **HTML5** – δομή και σημασιολογία
- **CSS3** – μεταβλητές, gradients, responsive grid
- **JavaScript (ES6)** – λογική αντίστροφης μέτρησης & ενημέρωση DOM
